# praiometro-backend
Backend para o app Praiômetro

## Back-end

1. Na máquina que deseja usar como servidor, rode o script `start_mongo.bat` (Windows) ou `start_mongo.sh` (Linux).
2. Inicie o virtual environment com `pyton -m venv venv`
3. Instale as dependências com `pip install -r requirements.txt`
3. Caso seja sua primeira vez rodando, popule o banco com `pyton popular_banco.py`
4. Rode `api_praiometro.py`
5. Rode `avaliador.py`
6. Rode `praiometro_hourly.py`
